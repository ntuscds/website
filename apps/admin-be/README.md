# Node - Express - Typescript admin backend server

## Getting Started

- Clone the repository

```
git clone https://github.com/ntuscse/website.git
```

- Install dependencies

```
cd ./apps/admin-be
yarn install
```

- Run the project directly in TS

```
yarn run dev
```

- Run tests

```
yarn test
```

## Running ESLint

```
yarn lint    // runs ESLint check + fix
```
